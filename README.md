## Usage

> Start the MongoDB server
> Seed Mongo data and start server by running `npm start`.
> Start the React application with `npm run dev:react`.
> Open browser at `127.0.0.1:4212` to see the application rendered.

## Requirements

- Node 6.13.0
- MongoDB 
- Mongoose

Optional
- Nodemon
